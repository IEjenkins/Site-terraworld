Установка PlasmoVoice

Обновленно : 1 Сентября 2023 г. 
    Гайд по установке PlasmoVoice
Если вы изпользуйте TLauncher :

Открыть его

В списке версий выбрать 1.20.1 ForgeOptifine ( Можно без Optifine )

Можно и Fabric 1.20.1



После нажимаем на папочку. Возле шестерёнки.

Открываеться проводник.



Открываем папку Mods

После сворачиваем. и начинаем качать сам мод.

Заходим сюда https://www.curseforge.com/minecraft/mc-mods/plasmo-voice/files



Скачиваем.

После перекидиваем файлик в папку Mods.

Запускаем/качаем клиент.

----
Для Official Minecraft Launcher
Если вы изпользуете Official Minecraft Launcher
----

Скачиваем мод также.

Но Forge/Fabric Устанавливаться отдельно.

Заходим в папку %APPDATA%

Для етого нажимаем Win+R



Нажимаем ок.

Выбираем roaming/.minecraft

Создаем/Заходим в папку Mods.

Скачиваем мод.

Закидиваем в папку.

После скачиваем клиент.


    Forge :  https://maven.minecraftforge.net/net/minecraftforge/forge/1.20.1-47.1.0/forge-1.20.1-47.1.0-installer.jar
    Fabric : https://minecraft-inside.ru/download/257496/
    + Fabric API : https://minecraft-inside.ru/download/282979/   
    
Кидаем в папку Mods

Установка Forge.

Запускаем скачанный файл



Всё. Ждем установки. и добавляем версию в лаунчер.

Установка Fabric

Перекидиваем Fabric API В папку Mods

После запускаем сам Fabric



После добавляем в лаунчер и запускаем.
